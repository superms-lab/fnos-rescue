# 第二阶段：飞牛 fnOS 原生版本

## 已完成

- fnOS、CPU架构和原生应用目录检测
- 在线服务清单与只读停用计划
- x86_64/ARM64通用原生 tar 安装包构建
- `/var/apps/fnos-rescue` 原生安装、升级备份和卸载还原
- 白名单 root helper，拒绝任意 shell 命令
- Debian 12 / fnOS 实机安装、运行、doctor、卸载和无残留验证

## 安全边界

- 默认只输出服务和挂载停用计划，不自动停止飞牛服务。
- 只有用户明确选择的源设备才能进入只读保护流程。
- 不允许把恢复结果写回源物理设备树。
- root helper 只接受固定 CLI 子命令，不接受 shell 或任意程序路径。
- 安装失败自动恢复旧版本；卸载可恢复升级前备份。

## 实机验收

在 Debian 12 x86_64 的 fnOS 主机完成：43项测试、原生包构建、安装、环境检测、doctor、
helper允许/拒绝测试、卸载和残留检查。所有媒体、下载、文件监控、SMB和NFS服务在测试前后
均保持 active，三块在线存储盘未参与测试。
